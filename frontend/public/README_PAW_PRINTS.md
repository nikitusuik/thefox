# Инструкция по добавлению изображения лапок

## Что нужно сделать:

1. Скопируйте изображение лапок (из предоставленного файла) в папку `frontend/public/`
2. Назовите файл `paw-prints.png`
3. Убедитесь, что файл доступен по пути `/paw-prints.png`

## Альтернативный вариант:

Если изображение имеет другое имя или формат, измените пути в `GamePage.vue`:
- Найдите все упоминания `/paw-prints.png`
- Замените на путь к вашему изображению

## Текущие использования:

- В секции "Opened clues" - отображаются лапки вместо текста подсказок
- В секции "Opened suspects" - отображаются лапки для подсказок подозреваемых
